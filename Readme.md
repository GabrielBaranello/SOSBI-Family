# A.I.S.O.
## What is it?
A.I.S.O. (asistente de instalacion de sistemas operativos in Spanish or operating system installation wizard in English) is an open source project that aims to create a tool for installing operating systems on computers. It still develoving but I think it has the potential to be a useful tool for users who want to install operating systems on their computers without having to go through the hassle of creating bootable USB drives or CDs.
## Features
- User-friendly interface: A.I.S.O. will have a simple and intuitive interface that will guide users through the installation process step by step.
- Support for multiple operating systems: A.I.S.O. will support the installation of various operating systems, including Windows, Linux, and macOS.
- Automated installation: A.I.S.O. will automate the download and creation of bootable media for the selected operating system, making the installation process faster and more efficient.
- Backup and restore: A.I.S.O. will allow users to create backups of their current operating system before installing a new one, and also provide an option to restore the previous operating system if needed.
## Contributing
You can contribute to the project by submitting pull requests, reporting issues, or suggesting new features. Your contributions are greatly appreciated and will help make A.I.S.O. a better tool for everyone.
## Use
To use A.I.S.O., simply run the application and follow the on-screen instructions. The tool will guide you through the process of selecting an operating system, downloading necessary files, and creating bootable media for installation.
