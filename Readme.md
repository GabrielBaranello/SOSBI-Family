# S.O.S.B.I. Wizard
#### Backup first. Install safe and simple.
## What is it?
S.O.S.B.I. Wizard (Safe Operating Sistem Backup & Install Wizard) is an open source project that aims to create a tool for installing operating systems on computers. It still develoving but I think it has the potential to be a useful tool for users who want to install operating systems on their computers without having to go through the hassle of creating bootable USB drives or CDs.
## Features
- User-friendly interface: S.O.S.B.I Wizard will have a simple and intuitive interface that will guide users through the installation process step by step.
- Support for multiple operating systems: S.O.S.B.I Wizard will support the installation of various operating systems, including Windows, Linux, and others.
- Automated installation: S.O.S.B.I Wizard will automate the download and creation of bootable media for the selected operating system, making the installation process faster and more efficient.
- Backup and restore: S.O.S.B.I Wizard will allow users to create backups of their current operating system before installing a new one, and also provide an option to restore the previous operating system if needed.
## Contributing
You can contribute to the project by submitting pull requests, reporting issues, or suggesting new features. Your contributions are greatly appreciated and will help make S.O.S.B.I Wizard a better tool for everyone.
## Use
To use S.O.S.B.I Wizard, simply run the application and follow the on-screen instructions. The tool will guide you through the process of selecting an operating system, downloading necessary files, and creating bootable media for installation.
